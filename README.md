# Thiên An Tea - Hệ thống bán trà online

**Thiên An Tea** là dự án website thương mại điện tử bán trà, bao gồm đầy đủ hệ thống:
- Giao diện người dùng (FE-USER)
- Giao diện quản trị (FE-ADMIN)
- Backend API (Laravel)

---

## Công nghệ sử dụng

### Frontend
- Next.js (React Framework)
- Axios (Gọi API)
- Tailwind CSS / Bootstrap (UI)

### Backend
- Laravel (RESTful API)
- PHP
- Middleware xác thực & phân quyền

### Database
- MySQL

### Công cụ hỗ trợ
- Postman (test API)
- Git & GitHub (quản lý mã nguồn)
- Figma (thiết kế giao diện)
